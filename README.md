# HomeVMLab

## Description
This repository contains hands-on home labs that simulate real-world IT tasks. The first lab focuses on **user management** in a Windows 10 virtual machine running on macOS. It demonstrates setting up administrator and standard user accounts, configuring permissions, and testing access control. These labs are designed to showcase fundamental IT support skills and build practical experience.

## Environment
- Host OS: macOS  
- Virtualization: VirtualBox  
- Guest OS: Windows 10  
- VM Specs: 50 GB storage, 6–8 GB RAM, 1–2 CPU cores
